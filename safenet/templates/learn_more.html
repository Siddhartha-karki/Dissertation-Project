Connected Network (Safety Insights)

Wi-Fi security is essential to keep your data safe from potential threats. Even if the network is encrypted, it is important to cross-check certain circumstances before continuing. It is highly recommended not to connect the publicy available Wi-Fi network with open connection type. Most of the devices shows a lock sign with the Wi-Fi name that means the Wi-Fi is secured where as the open network do not have those lockpad sign and sometimes might get auto connected when the auto-connect feature of device is enabled. Here are certain tips to stay vigilant while using Publicly Aviable Wi-Fi Networks:

  1. Check The Network Scan Report:

    What is in the report?

    The report consist of details like connected network name, Mac address which is also known as the physical address, Signal Strength, Encryption Type, Network Type, Port Scan, Access Point Detection

  2. Identify the Network
    • Check to See If the Network is encrypted or open.
    • In Case of open network, Verify the Captive portal Link with our Fake Captive portal detection tool. To do this copy the url link of the captive portal from the browser and paste it in the fake portal checker to begin the scan.
    • If the result is negative then avoid using the same network and look for alternative options.
  3. Open Ports:

    What are open ports

    An open port is basically a software value that identifies gateway or a network endpoint. Any TCP/IP communication made on the internet are done through the ports so it is important to keep these services secured as attackers can use misconfigured and vulnerabilities of ports as a backdoor to the device. While scanning if the network is open and there are open ports detected then we suggest to consider these points before browsing the internet.

    • Keep your device update as it contains security patches that address known vulnerabilities associated with open ports.
    • See for other alternative options if available, we strongly suggest not to share valuable information on this situation.
    • Always use strong authentication techniques like two factor authentication, Strong combination of passwords for any services associated with open ports to prevent unauthorized access.
  4. Signal Strength Considerations:

    Signal Strength of a Wi-Fi is not directly related with security implications or the presence of a rougue access points but it can indirectly impact security. To stay vigilant please follow these steps:

    • To identify the rogue access point users can check the signal strengths of available wifi networks. -30 dbm to -70 dbm as shown in the scanning result are the ideal signals for connectivity. A very strong signal over an unusually long range could be a sign of a fake access point( Wifi Name).
    • High Signal Strength: Caution against connecting to networks with excessively high signal strength, as it might indicate proximity to the access point. Turn off Auto-Connect.
    • Low Signal Strength can cause poor connectivity, always check the signal strength before connection.
  5. Fake Access Point Detection:

    What are Fake Access Points?

    A rogue access point is a malicious wireless access point set up by bad actors to mimic a legitimate network, tricking users into connecting to it. If the user is connected to these access points then the attacker can intercept, monitor, manipulate and even take over to the system leading to data theft and other security implications. To avoid connecting to Rogue Access Points and stay vigilant please cosider these steps:

    • Always Check if there are multiple Access points with similar WI-FI name, when found verify with the network provider for legitimate SSID. It is higly advised not to connect with the following networks without the proper authentication of network.
    • Check the scan result, if the connected network is detected as Fake Ap then disconnect the network and look for alternative option for communication.
    • Run your device through antivirus scanning just to check if the device is already compromised with malware payload and other attack.
    • Keep using the scanning tool time to time as fake access points are tricky and when auto-connect feature is on then the fake AP's can be auto connected having strong signal strength.
